# Links
A web application to showcase your links!

# Tech Stack
- [Vue.js](https://vuejs.org/)
- [Nuxt](https://nuxt.com/)
- [PHP](https://www.php.net/)
- [Laravel](https://laravel.com/)
- [MySQL](https://www.mysql.com/)
- [TailwindCSS](https://tailwindcss.com/)
